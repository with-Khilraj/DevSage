Before upgrading: Please backup your database and configuration files.

Migration steps:

  1. Run the database migration script: npm run migrate
  2. Update your configuration files to YAML format
  3. Update API endpoints in your applications to use /api/v2/
  4. Test thoroughly in a staging environment
